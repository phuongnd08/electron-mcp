Usage

Connecting with MCP Client

Set the following endpoint in applications like Claude Desktop:

http://localhost:3999/mcp

Note: Browser access is for information display only. Actual MCP communication is done via POST requests.

Available Tools

  • ping - Check server response
  • echo - Echo back messages
  • get_server_info - Get server information

Claude Desktop Configuration Example

{
  "mcpServers": {
    "electron-mcp-server": {
      "command": "curl",
      "args": ["-X", "POST", "http://localhost:3999/mcp"]
    }
  }
}
