使用方法

MCPクライアントでの接続

Claude Desktopなどのアプリケーションで、以下のエンドポイントを設定してください：

http://localhost:3999/mcp

利用可能なツール

  • ping - サーバーの応答確認
  • echo - メッセージのエコーバック
  • get_server_info - サーバー情報の取得

Claude Desktop設定例

{
  "mcpServers": {
    "electron-mcp-server": {
      "command": "curl",
      "args": ["-X", "POST", "http://localhost:3999/mcp"]
    }
  }
}
